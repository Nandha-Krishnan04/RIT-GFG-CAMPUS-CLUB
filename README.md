# RIT GFG Campus Club Platform 🚀

## Overview

The **RIT GFG Campus Club Platform** is a web-based platform designed to help students practice coding, learn programming concepts, and participate in campus club activities. The platform combines coding challenges, learning resources, leaderboards, and campus events to create an engaging learning environment for students.

## Game Mode 🎮

The **Game Mode** is the core feature of this platform. It transforms coding practice into a game-like experience where users progress through levels by solving programming challenges.

Users can choose topics such as **Arrays, Strings, or Graphs**, and solve problems based on difficulty levels:

* Beginner
* Intermediate
* Advanced

### Game Mode Features

**1. Level-Based Challenges**
Users start with beginner problems and unlock higher levels as they progress.

**2. Points System**
Each solved problem gives points. Harder problems reward more points.

**3. Debugging Challenges**
Some challenges provide **buggy code**, and users must find and fix the errors to solve the problem. This improves debugging skills and understanding of real coding issues.

**4. Unlock System**
New problems and levels unlock only after completing previous challenges, encouraging consistent learning.

**5. Progress Tracking**
The platform tracks the user's progress, showing how many problems they have completed and their current level.

**6. Leaderboard**
Users are ranked based on points earned, creating a competitive and motivating environment.

This gamified approach makes coding practice **interactive, competitive, and fun**, helping students improve their programming and problem-solving skills.
